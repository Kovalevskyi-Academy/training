# Задания для тренировки
## Изменим цветовую схему редактора
* В главном меню выберите `File → Settings → Color Theme → Monokai`

## Изучим основные команды Linux
* В терминале ниже введите команду 
```sh 
clear
```
* Выполните команду 
```sh
ls
```
* Выполните команду 
```sh
mkdir folder
```
* Повторите команду 
```sh
ls
```
